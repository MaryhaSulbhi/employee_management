# 🧑‍💼 Employee Management System in C

A console-based Employee Management System built using C programming language.  
This project demonstrates core concepts of system-level programming including pointers, dynamic memory allocation, linked lists, and file handling.

---

## 🚀 Features

- Add Employee
- Delete Employee
- Search Employee
- Display All Employees
- Persistent storage using binary file handling
- Dynamic memory management
- Modular programming structure

---

## 🛠 Technologies Used

- C Programming
- Linked List Data Structure
- Dynamic Memory Allocation (malloc, free)
- File Handling (fopen, fwrite, fread)
- Modular Design (.h and .c separation)
- Git & GitHub
